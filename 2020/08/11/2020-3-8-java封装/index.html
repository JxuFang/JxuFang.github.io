Java中的封装

发表于2020-08-11更新于2020-08-11

什么是封装

封装是将类的属性私有化并使用方法将其隐藏起来，控制用户对类的修改和访问程度。

为什么要封装

封装可以被当作是一种保护类的屏障，防止类的代码和数据轻易地被外部类随机访问或修改，从而维持程序的安全性。

如何实现封装

  1. 使属性私有化来限制对属性的访问。
  2. 为每个属性创建一对赋值方法（setter）和取值方法（getter），用于对属性的访问。
  3. 在赋值和取值方法中，加入对属性的存取和限制。

例题：编程找出m~n之间的所有完数，m和n为从键盘输入的两个整数
问题分析 ：题目的关键是完数。完数是一些特殊的自然数，这个数的所有真因子（除自身以外的约数）之和正好与它自身相等。
解决思路 ：要检验一个数是不是完数，首先要找到这个数除自身以外的约数，然后将这些约数相加与这个数进行比较即可。
如何找到一个数的约数呢？
约数：整数a除以整数b(b≠0) 除得的商正好是整数而没有余数，我们就说a能被b整除，或b能整除a。a称为b的倍数，b称为a的约数。由定义我们可以知道a除以b，余数为零，则b就是a的约数。
