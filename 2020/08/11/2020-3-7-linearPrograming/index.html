使用Matlab解决简单的线性规划问题

发表于2020-08-11更新于2020-08-11

如果你还没有Matlab，那么可以在这里安装！！！

什么是线性规划？

线性规划是一个在数学模型中能够得到最好结果的方法，而这个数学模型是用线性关系表示的。更正式地说，线性规化是一种优化线性目标函数的技术，这种技术受线性等式和线性不等式的约束。

线性规划能用来做什么?

在运筹学的领域，线性规划使用来得到最优解的。它可以让我们在一个问题的所有限制或约束条件中找到最好、最经济的解决办法。

问题引入

某机床生产甲、乙两种机床，每台销售后的利润分别为4千元、3千元。生产甲机床需要A，B机器加工，加工时间分别为每台2h和1h；生产乙机床需要A,B,C三种机器加工加工时间为每台各1h。若每天可用于加工的机器时数分别为A机器10h、B机器8h、C机器7h，问该生产甲、乙机床各几台，才能是总利润最大？

根据问题，我们假设需要生产甲机床x1x_1x1​台，乙机床x2x_2x2​台才能使利润最大。

max z = 4x1x_{1}x1​ + 3x2x_2x2​
2x1x_1x1​ + x2x_2x2​ ≤\leq≤ 10
x1x_1x1​ + x2x_2x2​ ≤\leq≤ 8
x1x_1x1​ ≤\leq≤ 7
x1x_1x1​ ,x2x_2x2​ ≥\geq≥ 0

以上就是根据题意得出的目标函数和约束条件。

线性规划的图解法
图解法可以说是使得问题更加的清晰和明了，正如数学教授华罗庚说的“数无形时少直觉，形少数时难入微”，对于简单的线性规划问题，图解法是一种好的解决办法。


给定一个固定z值，就会有一条相应的目标直线，而我们需要的是最大利润，那么就是希望z的值最大，当目标直线和可行域（阴影部分）某个点相交时使得z的值最大，那么这个点就是我们想要得到的可行解(2,6)

下面就是我们今天要说的 使用Matlab来求解线性规划问题

linprog是我们在matlab中解决线性规划问题所要用的函数，其中有几个重要的参数：

A 是一个线性不等式约束的M×\times×N系数矩阵，其中M是不等式的数量，N是自变量的数量。

b 是一个具有M个元素的列向量

这样的不等式应该是 A*x ≤\leq≤ b的形式。
For example：
x1x_1x1​ + 2x2x_2x2​ ≤\leq≤ 10
3x1x_1x1​ + 4x2x_2x2​ ≤\leq≤ 20
5x1x_1x1​ + 6x2x_2x2​ ≤\leq≤ 30
⇓\Downarrow⇓
A = [1,2;3,4;5,6];
b = [10;20;30];

Aeq 是一个线性等式约束的M×\times×N系数矩阵，其中M是不等式的数量，N是自变量>的数量。
beq 是一个具有M个元素的列向量
For example：
x1x_1x1​ + 2x2x_2x2​ +3x3x_3x3​ = 10
2x1x_1x1​ + 4x2x_2x2​ +x3x_3x3​= 20
⇓\Downarrow⇓
Aeq = [1,2,3;2,4,1];
beq = [10;20];

lb 是变量的下限（Lower bounds）
ub 是变量的上限（upper bounds）
For example：
-1 ≤\leq≤ x1x_1x1​ ≤\leq≤ 1.5
-0.5 ≤\leq≤ x2x_2x2​ ≤\leq≤ 1.25
⇓\Downarrow⇓
lb = [-1,-0.5];
ub = [1.5,1.25];
其中无穷的表示方法为 inf
f 是目标函数的系数向量
Example：f = [1,3,5,-6];

linprog的一些语法（Syntax）:

  1. x = linprog(f,A,b)
  2. x = linprog(f,A,b,Aeq,beq)
  3. x = linprog(f,A,b,Aeq,beq,lb,ub)

线性规划问题标准的矩阵表达形式为:

min CTC^TCTX
AX = b
X ≥\geq≥ 0
